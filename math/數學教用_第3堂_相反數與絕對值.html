第 3 堂課　相反數與絕對值

按數字 1、2、3… 逐步顯示步驟與解析 · 全民補習班 線上教學

概念 1　相反數

💡 概念說明

  • 在數線上，分居原點兩側、與原點 距離相等 的兩數，互為 相反數。例如 ＋4 與 −4。
  • 求相反數：只要把 性質符號改變（＋ 改成 −、− 改成 ＋）。
  • 0 的相反數是 0。
  • 一個數與它的相反數 相加，結果等於 0。
👨‍🏫 老師講解（4 題）
講 1＋9 的相反數是多少？
求相反數：只要把性質符號改變。
＋9 的「＋」改成「−」 → −9。
−9
講 2−5.3 的相反數是多少？
把性質符號改變：「−」改成「＋」。
−5.3 → ＋5.3。
＋5.3
講 3−6 的相反數是多少？
負數的相反數為正數：「−」改成「＋」。
−6 → ＋6。
驗證：（−6）＋（＋6）＝0 ✓。
＋6
講 4求 −2⅖ 的相反數。
數值不變，只改變性質符號。
−2⅖ 的「−」改成「＋」 → ＋2⅖。
＋2⅖
✏️ 學生練習（4 題）
練 1寫出相反數：＋7、−3、0、＋2.5、−⅗。
把每個數的性質符號改變；0 的相反數仍是 0。
＋7 → −7；−3 → ＋3；0 → 0。
＋2.5 → −2.5；−⅗ → ＋⅗。
−7、＋3、0、−2.5、＋⅗
練 2某數的相反數是 −5，這個數是多少？
相反數互相對應：把 −5 再取相反數就是原數。
−5 的相反數 → ＋5。
＋5
練 3在數線上標出 ＋3 與它的相反數。
＋3 在原點 0 的右邊，距離 3。
相反數 −3 與原點距離相等，在原點左邊距離 3。
＋3 在右 3；相反數 −3 在左 3
練 4−10 與 ＋10 是不是相反數？
判斷：數值相同、性質符號相反，且與原點距離相等。
−10 與 ＋10 數值都是 10、符號相反 → 是。
是

＋12 的相反數 → −12。

作 4一個數與它的相反數相加等於（　）。
相反數性質：數值相同、符號相反。
例：（＋5）＋（−5）＝0。

概念 2　絕對值

💡 概念說明

  • 一個數在數線上與 原點的距離，稱為這個數的 絕對值，記為「｜　｜」。
  • 正數的絕對值是它本身：｜＋5｜＝5。
  • 負數的絕對值是它的相反數：｜−5｜＝5。
  • 0 的絕對值是 0：｜0｜＝0。
  • 絕對值代表距離，所以 絕對值不會是負數。
👨‍🏫 老師講解（4 題）
講 1求 ｜＋8｜。
絕對值＝與原點的距離，正數的絕對值是它本身。
｜＋8｜＝8。
8
講 2求 ｜−6｜。
負數的絕對值是它的相反數（去掉負號）。
｜−6｜＝6。
6
講 3求 ｜＋11｜。
正數的絕對值是它本身。
｜＋11｜＝11。
11
講 4求 ｜−3.5｜。
負數的絕對值是它的相反數，把負號去掉。
｜−3.5｜＝3.5。
3.5
✏️ 學生練習（4 題）
練 1求值：｜＋5｜、｜−9｜、｜0｜、｜−2.5｜。
正數取本身、負數去負號、0 取 0。
｜＋5｜＝5；｜−9｜＝9。
｜0｜＝0；｜−2.5｜＝2.5。
5、9、0、2.5
練 2絕對值是 7 的數有哪些？
絕對值＝與原點距離 7，原點兩側各有一個。
距離 7 的數有 ＋7 與 −7。
＋7 與 −7
練 3求 ｜−⅔｜。
負數的絕對值是它的相反數，去掉負號。
｜−⅔｜＝⅔。
⅔
練 4一個負數的絕對值是 4，這個數是多少？
絕對值 4 的數有 ＋4 與 −4。
題目限定是「負數」 → −4。
−4

作 1｜＋12｜＝（　）。
正數的絕對值是它本身。
｜＋12｜＝12。
12
作 2｜−15｜＝（　）。
負數的絕對值是它的相反數，去掉負號。
｜−15｜＝15。

作 3｜0｜＝（　）。
0 與原點的距離是 0。
｜0｜＝0。
0
作 4絕對值等於 10 的數有（　）。
與原點距離 10 的數，原點兩側各一個。
有 ＋10 與 −10。
＋10 與 −10
作 5｜−4.8｜＝（　）。
負數的絕對值是它的相反數，去掉負號。
｜−4.8｜＝4.8。
4.8
作 6絕對值最小的數是（　）。
絕對值代表距離，不會是負數，最小為 0。
絕對值為 0 的數是 0。
0
概念 3　絕對值的運算與比大小

💡 概念說明

  • 含絕對值的計算：先算出每一個絕對值（變成一般的數），再做加減。
  • 例：｜−3｜＋｜＋2｜＝3＋2＝5。
  • 例：｜−7｜−｜−5｜＝7−5＝2。
  • 比較大小時，也 先求出各個絕對值再比較。
👨‍🏫 老師講解（4 題）
講 1計算 ｜−8｜＋｜＋1｜。
先求各個絕對值：｜−8｜＝8、｜＋1｜＝1。
再相加：8＋1＝9。
8＋1＝9
講 2計算 ｜−9｜−｜−4｜。
先求各個絕對值：｜−9｜＝9、｜−4｜＝4。
再相減：9−4＝5。
9−4＝5
講 3計算 ｜−6｜＋（−10）。
先求絕對值：｜−6｜＝6。
代回算式：6＋（−10）。
6＋（−10）＝−4。
6＋（−10）＝−4
講 4比較 ｜−11｜、｜＋6｜、｜−1｜ 由大到小。
先求各個絕對值：｜−11｜＝11、｜＋6｜＝6、｜−1｜＝1。
比較 11＞6＞1。
所以 ｜−11｜＞｜＋6｜＞｜−1｜。
｜−11｜＞｜＋6｜＞｜−1｜（11＞6＞1）
✏️ 學生練習（4 題）
練 1計算 ｜−8｜＋｜＋3｜。
先求各個絕對值：｜−8｜＝8、｜＋3｜＝3。
再相加：8＋3＝11。
8＋3＝11
練 2計算 ｜＋10｜−｜−6｜。

再相減：10−6＝4。
10−6＝4
練 3計算 ｜−9｜−｜＋9｜。
先求各個絕對值：｜−9｜＝9、｜＋9｜＝9。
再相減：9−9＝0。
9−9＝0
練 4比較 ｜−2｜、｜＋5｜、｜−7｜ 由大到小。
先求各個絕對值：｜−2｜＝2、｜＋5｜＝5、｜−7｜＝7。
比較 7＞5＞2。
所以 ｜−7｜＞｜＋5｜＞｜−2｜。
｜−7｜＞｜＋5｜＞｜−2｜
