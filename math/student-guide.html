概念 1 正數與負數
📘 概念說明
  • 在日常生活中，意義「相對或相反」的量，都可以用帶有「＋（讀作正）」和「－（讀作負）」的數來表示。
  • 用正負號表示溫度：攝氏 30 度記為 ＋30 度；零下 10 度記為 −10 度。
  • 用正負號表示方向：以基準點為 0，規定一邊為正、另一邊為負。如東為正、西為負，向東 2 公里記為 ＋2 公里，向西 3 公里記為 −3 公里。
  • 用正負號表示賺賠：賺 35000 元記為 ＋35000 元；賠 2300 元記為 −2300 元。
  • 正數：凡是大於 0 的數（如 ＋5、＋3.5、＋¾）。負數：凡是小於 0 的數（如 −5、−3.5、−¾）。
  • 0 既不是正數，也不是負數。

概念 2 整數
📘 概念說明
  • 整數包括：正整數、零、負整數。
  • 正整數：1、2、3、4…，又稱為「自然數」。

概念 3 數線三要素與整數的畫法
📘 概念說明
  • 數線有三個要素：原點、正向、單位長。
  • 原點：在直線上任取一點 O 當基準點，代表數字 0。
  • 正向：通常規定向右為正向（在右端加箭號）；原點右邊的點是正數，左邊的點是負數。
  • 單位長：在原點右方選取一段定長當作一個單位。
  • 整數的畫法：自原點向右，每隔一個單位長取一點，依序標 1、2、3、…；向左每隔一個單位長取一點，依序標 −1、−2、−3、…。

概念 1 分數、小數在數線上的畫法
📘 概念說明
  • 整數以外的數，可以把一個單位長「等分」後，再數格子標出對應的點。
  • 分數：如 ½ 把 0 到 1 等分成 2 段，取第 1 段的端點；−1½ 在 −1 與 −2 的中間。
  • 小數：如 2.5 在 2 與 3 的中間；−0.5 在 0 與 −1 的中間。
  • 畫法重點：先看是幾分之幾或幾點幾，決定要把單位長等分成幾段，再從整數點往正向或負向數格子。

概念 2 有理數
📘 概念說明
  • 整數、分數、小數都能在數線上找到對應的點，這些數統稱為「有理數」。
  • 整數可以看成分母是 1 的分數（如 3 ＝ 3/1），所以整數也是有理數。
  • 分數、有限小數都是有理數。
  • 每一個有理數，在數線上都有一個對應的點。

概念 3 數的大小比較
📘 概念說明
  • 在數線上，越靠右邊的數越大：右邊的數 ＞ 左邊的數。
  • 正數 ＞ 0 ＞ 負數。
  • 兩個正數：絕對值（數字）大的比較大。
  • 兩個負數：絕對值（數字）大的反而比較小，例如 −7 ＜ −3。

三個概念：相反數 ・ 絕對值 ・ 絕對值的運算與比大小
概念 1 相反數
📘 概念說明
  • 在數線上，分居原點兩側、與原點「距離相等」的兩數，互為相反數。例如 ＋4 與 −4。
  • 求相反數：只要把性質符號改變（＋ 改成 −、− 改成 ＋）。
  • 0 的相反數是 0。
  • 一個數與它的相反數相加，結果等於 0。

概念 2 絕對值
📘 概念說明
  • 一個數在數線上與原點的距離，稱為這個數的「絕對值」，記為「｜　｜」。
  • 正數的絕對值是它本身：｜＋5｜＝5。
  • 負數的絕對值是它的相反數：｜−5｜＝5。
  • 0 的絕對值是 0：｜0｜＝0。
  • 絕對值代表距離，所以絕對值不會是負數。

三個概念：同號數相加 ・ 異號數相加 ・ 運算律與加法應用
概念 1 同號數相加
📘 概念說明
  • 同號數相加：把兩數的絕對值相加，和的前面加上與原來相同的符號。
  • 正數加正數：3＋4＝7。
  • 負數加負數：(−5)＋(−2)＝−(5＋2)＝−7。
  • 兩個正數的和一定是正數；兩個負數的和一定是負數。

概念 2 異號數相加（含相反數、與 0 相加）
📘 概念說明
  • 異號數相加：把兩數的絕對值相減，和的前面加上絕對值較大者的符號。
  • 例：5＋(−2)＝＋(5−2)＝3；(−4)＋3＝−(4−3)＝−1。
  • 相反數相加等於 0：(−6)＋6＝0。
  • 任何數與 0 相加，值不變：5＋0＝5、0＋(−7)＝−7。

概念 3 加法運算律與加法應用
📘 概念說明
  • 交換律：甲＋乙＝乙＋甲。
  • 結合律：(甲＋乙)＋丙＝甲＋(乙＋丙)。
  • 連加：三個以上的數相加可任意次序，常把同號的、或能湊成整十整百的先加，計算較快。
  • 加法應用：賺賠（賺正、賠負）、方向位移（一向為正、反向為負）等問題，把情境翻成正負數再相加。

整數的減法、加減混合、總複習
三個概念：整數的減法 ・ 加減混合計算 ・ 加減法與絕對值
概念 1 整數的減法
📘 概念說明
  • 整數的減法：甲減乙 ＝ 甲加上乙的相反數。也就是「減一個數，等於加它的相反數」。
  • 正數減正數：7−2＝7＋(−2)＝5。
  • 正數減負數：9−(−2)＝9＋2＝11。
  • 負數減正數：(−5)−4＝(−5)＋(−4)＝−9。
  • 負數減負數：(−8)−(−3)＝(−8)＋3＝−5。

2計算 14−(−20)。
3計算 (−10)−13。
4計算 (−16)−(−12)。

概念 2 加減混合計算
📘 概念說明
  • 加減混合：先把括號去掉，再由左到右依序計算。
  • 去括號規則：括號前是「＋」，括號內不變號；括號前是「−」，括號內每一項都變號。
  • 例：162−(−45)＋32 ＝ 162＋45＋32 ＝ 239。
  • 也可以善用交換律、結合律，先湊整再算。

概念 3 加減法與絕對值
📘 概念說明
  • 含絕對值的加減：先算出絕對值內的式子（得到一個數），再取絕對值，最後做外面的加減。
  • 絕對值「｜　｜」就像一個括號，要先把裡面算完。
  • 例：｜3＋(−19)＋7｜＝｜−9｜＝9。
  • 例：｜3＋(−19)＋7｜−｜(−17)−5＋43｜＝9−21＝−12。
