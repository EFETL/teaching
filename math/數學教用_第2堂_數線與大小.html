💡 概念 1　分數、小數在數線上的畫法

  • 整數以外的數，可以把一個單位長 「等分」 後，再數格子標出對應的點。
  • 分數：如 ½ 把 0 到 1 等分成 2 段，取第 1 段的端點；−1½ 在 −1 與 −2 的中間。
  • 小數：如 2.5 在 2 與 3 的中間；−0.5 在 0 與 −1 的中間。
  • 畫法重點：先看是幾分之幾或幾點幾，決定要把單位長 等分成幾段，再從整數點往正向或負向數格子。

💡 概念 2　有理數

  • 整數、分數、小數都能在數線上找到對應的點，這些數統稱為 「有理數」。
  • 整數可以看成 分母是 1 的分數（如 3 ＝ 3/1），所以整數也是有理數。
  • 分數、有限小數都是有理數。
  • 每一個有理數，在數線上都有 一個對應的點。

💡 概念 3　數的大小比較

  • 在數線上，越靠 右邊 的數越大：右邊的數 ＞ 左邊的數。
  • 正數 ＞ 0 ＞ 負數。
  • 兩個正數：絕對值（數字）大的比較大。
  • 兩個負數：絕對值（數字）大的反而比較小，例如 −7 ＜ −3。
👨‍🏫 概念 3　老師講解（4 題）
講 1用數線比較 −4 與 ＋1 的大小。
−4 在原點左邊、＋1 在原點右邊。
數線上越右邊越大，＋1 在右邊。
所以 −4 ＜ ＋1。
−4 ＜ ＋1（＋1 在右邊較大）
講 2比較 0 與 −6；比較 ＋5 與 0。
口訣：正數 ＞ 0 ＞ 負數。
−6 是負數，比 0 小 → 0 ＞ −6。
＋5 是正數，比 0 大 → ＋5 ＞ 0。
0 ＞ −6；＋5 ＞ 0
講 3比較兩個負數 −2 與 −9。
兩個都是負數，先看絕對值：｜−2｜＝2、｜−9｜＝9。
負數中 絕對值大的反而小，9 ＞ 2，所以 −9 較小。
所以 −2 ＞ −9（−2 在數線右邊）。
−2 ＞ −9
講 4將 −1、＋4、0、−6 由小到大排列。
先分類：負數 −1、−6；零 0；正數 ＋4。
兩負數比大小：絕對值大的小 → −6 ＜ −1。
負 ＜ 0 ＜ 正，依序排出 −6 ＜ −1 ＜ 0 ＜ ＋4。
−6 ＜ −1 ＜ 0 ＜ ＋4
