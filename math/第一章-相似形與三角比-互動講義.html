💡 核心觀念：連比例

  • 連比 a:b:c 各項同乘或同除相同數，比值不變。
  • 求連比（LCM 法）：找共同項的最小公倍數，把兩個比湊成同一個中間項，再合併。
  • 若 ax = by，則 x:y = b:a（係數對調）。
  • 化最簡整數比：先清除小數或分數，再除以各項 GCD。
  • 三個比連接時，中間項必須統一才能直接合併成連比。
👨‍🏫 老師講解（8 題）
講 1已知 x:y = 2:3，y:z = 3:5，求 x:y:z。
確認條件：x:y = 2:3，y:z = 3:5，中間項 y 在兩個比中都是 3，已經相同。
列性質：中間項相同時，可以直接合併成連比 x:y:z。
代入計算：x:y = 2:3，y:z = 3:5，直接合併 → x:y:z = 2:3:5。
得出答案：x:y:z = 2:3:5。
x:y:z = 2:3:5
講 2已知 a:b = 3:4，b:c = 6:7，求 a:b:c。
確認條件：a:b = 3:4，b:c = 6:7，中間項 b 分別是 4 和 6，不相同，需要用 LCM 法統一。
列性質：找 4 和 6 的最小公倍數 LCM(4,6) = 12，把 b 都化成 12。
代入計算：a:b = 3:4 → 乘以 3 → 9:12；b:c = 6:7 → 乘以 2 → 12:14。合併得 a:b:c = 9:12:14。
得出答案：a:b:c = 9:12:14。
a:b:c = 9:12:14

確認條件：x:y = 5:3，y:z = 4:5，中間項 y 分別是 3 和 4，不相同，需要 LCM 法。
列性質：LCM(3,4) = 12，把兩個比中的 y 都化成 12。
代入計算：x:y = 5:3 → 乘以 4 → 20:12；y:z = 4:5 → 乘以 3 → 12:15。合併得 x:y:z = 20:12:15。
得出答案：x:y:z = 20:12:15。
x:y:z = 20:12:15

確認條件：3x = 4y → 用係數對調性質求 x:y；y:z = 2:5 已知。
列性質：ax = by → x:y = b:a，所以 3x = 4y → x:y = 4:3。
代入計算：x:y = 4:3，y:z = 2:5。LCM(3,2) = 6。x:y = 8:6；y:z = 6:15。合併得 x:y:z = 8:6:15。
得出答案：x:y:z = 8:6:15。

講 5已知 a:b = 1/2 : 1/3，b:c = 0.4:0.6，化為最簡整數連比 a:b:c。
確認條件：a:b = 1/2 : 1/3（分數比），b:c = 0.4:0.6（小數比），需先各自化為整數比。
列性質：分數比同乘最小公倍母數；小數比同乘 10（去小數點），再除以 GCD。
代入計算：a:b = 1/2 : 1/3 → 乘以 6 → 3:2；b:c = 0.4:0.6 → 乘以 10 → 4:6 → 除以 2 → 2:3。LCM(2,2)=2，兩邊中間項已是 2:2，合併 a:b:c = 3:2:3。
得出答案：a:b:c = 3:2:3。
a:b:c = 3:2:3
講 6甲、乙、丙三人分紅，甲:乙 = 5:3，乙:丙 = 4:9。若三人分紅總額為 34000 元，甲分得多少元？
確認條件：甲:乙 = 5:3，乙:丙 = 4:9，需先求連比再依比分配。
列性質：LCM(3,4) = 12，把乙統一成 12。甲:乙 = 20:12；乙:丙 = 12:27。連比 = 20:12:27，總份數 = 59。
代入計算：甲的份數 = 20，甲分得 = 34000 × 20/59 = 680000/59。59 × 11 = 649，59 × 12 = 708，不整除，確認：20+12+27=59，34000÷59 不整除，改用 34000×20÷59。34000/59 ≈ 576.27，不整除，故答案以分數表示 = 680000/59 ≈ 11525 元。（提示：本題若用 a:b=5:3, b:c=4:9，LCM(3,4)=12，a:b=20:12, b:c=12:27，總59份，34000÷59×20 = 680000/59。）
得出答案：甲分得 680000/59 ≈ 11525 元（精確值 680000/59 元）。
甲分得 680000/59 元（約 11525 元）

💡 核心觀念：連比例式的應用

  • 連比轉化：x:y:z = a:b:c 等價於設 x=ar, y=br, z=cr（r≠0），再利用已知條件求 r。
  • 連比求各量：建立方程（如總和 x+y+z=某數，或差的條件），解出 r 後各自代入。
  • 等積連比：若 ax = by = cz = k，則 x:y:z = 1/a : 1/b : 1/c（即 bc : ac : ab）。
  • 按比分配：總量 S 按 a:b:c 分配時，各份 = S × a/(a+b+c)、S × b/(a+b+c)、S × c/(a+b+c)。
  • 生活應用：食譜、費用、工作量等按比例分配問題，均可先設公比 r，列方程求解。

步驟1｜確認條件：x:y = 3:5，x+y = 24。
步驟2｜設公比：令 x = 3r，y = 5r（r≠0）。
步驟3｜代入求解：3r + 5r = 24 → 8r = 24 → r = 3，∴ x = 9，y = 15。
步驟4｜驗算：9+15 = 24 ✓，9:15 = 3:5 ✓。

步驟2｜按比分配公式：各份 = 總量 × 自身比值 ÷ 比值總和（4+3+2=9）。
步驟3｜代入計算：甲 = 1800×4/9 = 800，乙 = 1800×3/9 = 600，丙 = 1800×2/9 = 400。

💡 核心觀念：等高三角形

  • 等高三角形面積比＝底邊長之比：若兩個三角形的高相同，則面積比等於底邊長之比。
  • 同一頂點 A，△ABD 與 △ACD 共頂點，∴ S△ABD : S△ACD = BD : DC。
  • 兩平行線間的三角形：底邊在同一直線上，頂點都在另一條平行線上，高相同，面積比＝底邊比。
  • 計算技巧：設面積為 kr（k 為倍數，r 為比值單位），利用已知總面積或差求 k。
  • 推廣應用：梯形中，對角線將梯形切成四個三角形，兩腰上的三角形面積相等（同底同高）。

💡 核心觀念：平行線截比例線段

  • 三角形中位線比例：△ABC 中，P 在 AB、Q 在 AC，若 PQ∥BC，則 AP:PB = AQ:QC（分段比相等）。
  • 整體比性質：PQ∥BC 時，AP:AB = AQ:AC = PQ:BC（小三角形與大三角形對應邊成比例）。

  • 常用逆定理：若 AP:PB = AQ:QC，則 PQ∥BC（可用來判斷兩線是否平行）。

💡 核心觀念：利用比例線段判別平行

  • △ABC 中，P 在 AB、Q 在 AC，若 AP:PB = AQ:QC（分段比相等），則 PQ∥BC。
  • 等價條件：若 AP:AB = AQ:AC（整體比相等），則 PQ∥BC。
  • 陷阱警示：Q 必須在 AC 上；若 Q 在 BC 上，AP:PB = AQ:QR 不能判斷 PR∥BC。
  • 判別平行的三步驟：① 確認兩點分別在哪兩邊 → ② 化簡比值 → ③ 比值相等即平行。
  • 比例可互換：AP:PB = AQ:QC ⟺ AP/AB = AQ/AC ⟺ PB/AB = QC/AC，三種寫法等價。

💡 核心觀念：三角形兩邊中點連線段性質

  • 中點連線段定理：△ABC 中，D、E 分別為 AB、AC 的中點，則 DE ∥ BC 且 DE = ½ BC。
  • 逆定理：若 DE ∥ BC 且 D 為 AB 中點，則 E 必為 AC 中點（E 平分 AC）。
  • 連續取中點：每往內一層，連線段長度乘以 ½；第 n 層連線段 = 原底邊 × (½)ⁿ。
  • 梯形中位線定理：梯形兩腰中點連線 ∥ 兩底，長度 = (上底 + 下底) ÷ 2。

💡 核心觀念：圖形的縮放

  • 以 O 為縮放中心，縮放比 k：新頂點 = O 點出發，沿 O→原頂點方向，距離乘以 k。
  • 縮放後對應線段互相平行（或共線），長度比等於縮放比 k。
  • 縮放不改變角度大小，縮放前後兩圖形相似，相似比 = k（或 1/k）。
  • k > 1 為放大；0 < k < 1 為縮小；k < 0 則先做中心對稱再縮放。
  • O 在圖形外時，新頂點在 O→原頂點射線的延長線上，離 O 的距離為原距離 × k。

💡 核心觀念：三角形的相似性質

  • AA 相似：兩組對應角相等，則兩三角形相似（最常用，平行線夾角、對頂角均可用）。
  • SAS 相似：一組夾角相等，且夾角兩側對應邊成比例，則兩三角形相似。
  • SSS 相似：三組對應邊兩兩成比例，則兩三角形相似。
  • 相似三角形的對應邊比（相似比）相等，對應角相等，面積比等於相似比的平方。
  • 平行線截三角形：△ABC 中 PQ∥BC，則 △APQ∼△ABC（AA），且 AP/AB = AQ/AC = PQ/BC。

  • 相似比 k：△ABC∼△A'B'C'，對應邊之比 = k（相似比）

💡 核心觀念：簡易測量

  • 影子法（同一時刻）：直立物高度之比 = 影長之比，即 物高 / 竿高 = 物影長 / 竿影長（共同仰角 + 直角，AA 相似）
  • 鏡射法：眼高 / 物高 = 眼到鏡距 / 物到鏡距（入射角 = 反射角，AA 相似）
  • AA 相似判定：兩三角形各有兩組角相等，則兩三角形相似；相似三角形對應邊成比例
  • 列比例式步驟：① 確認兩個相似△ ② 說明 AA 理由 ③ 對應邊寫成比例 ④ 交叉相乘求解
  • 單位一致：代入比例式前，所有長度須換成同一單位，答案再換回題目要求的單位

💡 核心觀念：三角比

  • 三角比定義：直角△ABC，∠C＝90°，∠A＝θ，則 sin θ＝BC／AB（對邊÷斜邊），cos θ＝AC／AB（鄰邊÷斜邊），tan θ＝BC／AC（對邊÷鄰邊）。
  • 特殊角值：sin30°＝1/2、cos30°＝√3/2、tan30°＝√3/3；sin45°＝√2/2、cos45°＝√2/2、tan45°＝1；sin60°＝√3/2、cos60°＝1/2、tan60°＝√3。
  • 畢氏恆等式：對任意銳角 θ，sin²θ＋cos²θ＝1，此式是許多計算題的關鍵工具。
  • 商數關係：tan θ＝sin θ ÷ cos θ，利用這個關係可以在已知兩個三角比時求出第三個。
  • 邊長互求：若斜邊為 c、∠A＝θ，則對邊＝c·sin θ、鄰邊＝c·cos θ；或已知一邊求其他邊時，靈活選用 sin、cos、tan。
