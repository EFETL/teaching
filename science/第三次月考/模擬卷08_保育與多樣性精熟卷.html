1.生物分類的階層由大到小排列，下列何者正確？分類基礎
A界 → 綱 → 門 → 科 → 目 → 屬 → 種
B界 → 門 → 綱 → 目 → 科 → 屬 → 種
C種 → 屬 → 科 → 目 → 綱 → 門 → 界
D門 → 界 → 綱 → 目 → 科 → 種 → 屬
【答案】B　【解析】cls：分類階層由大到小為界、門、綱、目、科、屬、種，種是最基本單位；越下層親緣越近、種類越少。其餘排序錯誤。(課本對應)
2.關於林奈的「二名法」，下列敘述何者正確？分類基礎
A學名由綱名加目名組成
B學名一律用中文書寫且不分大小寫
C同屬的兩種生物親緣關係通常較遠
D學名由屬名加種小名組成，以拉丁文斜體書寫、屬名首字母大寫
【答案】D　【解析】cls：二名法學名＝屬名＋種小名，拉丁文斜體、屬名首字大寫；同屬生物親緣較近。學名非綱目組成、非中文、同屬非較遠。(課本對應)
3.下列哪一類植物「最早演化出維管束」，有真正的根莖葉、但以孢子繁殖、無種子？植物界
A被子植物
B蘚苔植物
C裸子植物
D蕨類植物
【答案】D　【解析】plant：蕨類最早演化出維管束、有真正根莖葉、以孢子繁殖、無種子。蘚苔無維管束，裸子與被子以種子繁殖。(課本對應)
4.關於蘚苔植物（如地錢、土馬騌）的敘述，下列何者正確？植物界
A會開花並結出包住種子的果實
B具發達維管束且植株高大
C以毬果繁殖且受精不需水
D無維管束、植株矮小、受精時需以水為媒介，是最早登陸的植物
【答案】D　【解析】plant：蘚苔無維管束、矮小、生陰濕處、受精需水，是最早登陸的植物。發達維管束是蕨類以上，毬果是裸子，開花結果是被子。(課本對應)
5.裸子植物與被子植物最主要的差異在於下列何者？植物界
A裸子植物無維管束，被子植物才有維管束
B裸子植物有花有果，被子植物無花無果
C裸子植物胚珠裸露、無花無果實，被子植物有花且種子被果實包住
D兩者都以孢子繁殖、無種子
【答案】C　【解析】plant：裸子胚珠裸露、無真正花與果實、以毬果生殖；被子有花、種子由子房發育成的果實包住。兩者皆有種子與維管束。(課本對應)
6.下列關於單子葉與雙子葉植物的比較，何者正確？植物界
A兩者花瓣數目都固定為 7 片
B單子葉葉脈多為平行脈、根多為鬚根；雙子葉葉脈多為網狀脈、根多為軸根
C單子葉子葉有兩片、雙子葉子葉有一片
D單子葉葉脈為網狀脈、雙子葉為平行脈
【答案】B　【解析】plant：單子葉—子葉1、鬚根、平行脈、花瓣3或3倍數；雙子葉—子葉2、軸根、網狀脈、花瓣4/5倍數。選項把特徵對調或寫錯。(課本對應)

A棘皮動物

【答案】C　【解析】inv：水螅、水母、海葵、珊瑚具刺絲胞捕食、輻射對稱、有口無肛門，屬刺絲胞動物。軟體柔軟不分節、環節分節、棘皮具管足，皆不符。(課本對應)
8.昆蟲、蝦蟹、蜘蛛與蜈蚣共同具有的特徵是下列何者？牠們也是動物界中種類最多的一門。無脊椎動物
A具外骨骼、身體與附肢分節、會蛻皮
B體表具管足、輻射對稱

D身體柔軟不分節、具外套膜

9.海星、海膽、海參都僅生活於海洋，並具有特殊的運動構造，下列何者為其共同特徵？無脊椎動物
A具刺絲胞捕食、可行出芽生殖
B身體分成許多相似的環節、具剛毛

D具管足、體表多有棘、成體多為五輻對稱
【答案】D　【解析】inv：棘皮動物具管足、體表有棘、成體五輻對稱、僅生活於海洋。刺絲胞是腔腸，環節有剛毛，外套膜是軟體，皆非棘皮特徵。(課本對應)

A體表具管足且為輻射對稱
B身體柔軟、不分節，多具外套膜，有的有殼有的無殼

11.關於爬蟲類「比兩生類更能適應陸地生活」的原因，下列敘述何者正確？脊椎動物
A成體必須以鰓呼吸
B體表裸露濕潤、無法防止水分散失
C體內受精且卵具殼、體表有鱗片或骨板可減少水分散失，用肺呼吸

【答案】C　【解析】vert：爬蟲類體內受精、卵具殼、不必回水中生殖，體表鱗片/骨板防水、用肺呼吸，故更適應陸地。卵無殼、鰓呼吸、體表裸露皆為兩生類特徵。(課本對應)
12.下列哪一種動物雖名為「魚」或常被誤認，實際上卻是哺乳類？脊椎動物
A海豚
B彈塗魚

13.關於鳥類能適應飛行的構造特徵，下列敘述何者「錯誤」？脊椎動物
A瞬膜是飛行的主要動力來源
B具氣囊輔助呼吸，可進行雙重呼吸
C前肢特化為翼、體表覆有羽毛

【答案】A　【解析】vert：瞬膜是保護眼睛的構造，與飛行動力無直接關係，故為錯誤敘述。骨骼中空、氣囊雙重呼吸、前肢成翼與羽毛才是飛行適應。(課本對應)

16.關於「族群」與「群集」的定義，下列敘述何者正確？族群與群集

【答案】A　【解析】pop：族群＝同時間同區域同種所有個體；群集＝同區域所有不同種族群的總合。加上非生物環境才是生態系，選項把定義對調或混入環境。(課本對應)

A160
B2000
C500
D400
【答案】D　【解析】pop：標識再捕法總數＝標記數×再捕總數÷再捕中標記數＝80×100÷20＝400 尾。其餘為錯誤運算（如未除、除錯數）。(課本對應)

20.關於「環境負荷量（容納量）」與 S 型成長曲線的敘述，下列何者正確？（如圖）族群與群集

21.豆科植物的根部與根瘤菌共同生活，根瘤菌可固定氮供植物利用、植物則提供根瘤菌養分與棲所，此種雙方皆受益的關係屬於？交互作用
A互利共生
B寄生
C片利共生
D掠食
【答案】A　【解析】inter：雙方皆得利（＋／＋）為互利共生。片利是一益一無影響，寄生是一益一害，掠食是捕食者得利、被食者死亡，皆不符合。(課本對應)
22.鮣魚以吸盤附著在鯊魚腹部，藉鯊魚移動取得搬運與殘食碎屑，對鯊魚卻幾乎沒有影響。這種關係屬於？交互作用
A競爭
B片利共生
C寄生
D互利共生
【答案】B　【解析】inter：一方（鮣魚）受益、另一方（鯊魚）無明顯影響（＋／0）為片利共生。互利雙方皆益，寄生宿主受害，競爭雙方皆受影響。(課本對應)
23.菟絲子纏繞在其他植物上，以特化構造插入宿主吸取養分，使宿主生長受抑制。這種關係屬於？交互作用
A片利共生
B互利共生
C掠食
D寄生
【答案】D　【解析】inter：菟絲子得利、宿主受害（＋／−）為寄生。互利雙方皆益，片利一方無影響，掠食通常指捕食者直接吃掉被食者個體。(課本對應)
24.在同一塊田中混種甲、乙兩種需要相同陽光與養分的植物後，兩者的生長量都比單獨種植時下降。甲、乙之間最可能的關係是？交互作用
A互利共生
B掠食
C競爭
D片利共生
【答案】C　【解析】inter：兩者爭奪相同有限資源、雙方皆受抑制（−／−）為競爭。互利、片利皆有受益方，掠食是一方吃另一方，混養同降是典型競爭。(課本對應)
25.關於「掠食（捕食）」關係，下列敘述何者正確？交互作用
A掠食關係對被食者有利、對捕食者有害
B捕食者數量與獵物數量永遠完全相等
C捕食者數量與獵物數量常呈交錯的週期性波動
D捕食者與獵物雙方都受益
【答案】C　【解析】inter：山貓-雪鞋兔模式中，獵物多→捕食者增，捕食者多→獵物減，兩者數量交錯週期波動。掠食是捕食者得利、被食者受害（＋／−）。(課本對應)
26.農民釋放瓢蟲來吃蚜蟲、放養蓋斑鬥魚吃孑孓以控制病媒蚊，這類利用天敵控制有害生物的方法稱為？交互作用
A生物防治
B優養化
C互利共生
D群集演替
【答案】A　【解析】inter：利用天敵（掠食/寄生）控制有害生物為生物防治，可減少農藥。互利共生不會用來防治，優養化是水質問題，演替是群集隨時間變化。(課本對應)
27.關於生物防治，下列敘述何者「錯誤」？交互作用
A只要是互利共生的生物都能拿來進行生物防治
B平腹小蜂可用來防治荔枝椿象
C引入外來天敵前須謹慎評估，以免變成外來入侵種
D生物防治可減少化學農藥的使用
【答案】A　【解析】inter：生物防治利用掠食/寄生/競爭等天敵，互利共生對雙方皆有利、無法抑制害蟲，故「互利共生都能防治」錯誤。其餘三者皆正確。(課本對應)
28.在大多數生態系中，能量最終的來源是下列何者？能量流動
A土壤中的礦物質
B分解者的呼吸作用
C岩石風化釋放的熱
D太陽（陽光）
【答案】D　【解析】energy：太陽是大多數生態系最終能量來源，生產者行光合作用把光能轉成化學能。深海熱泉的硫化菌是少數例外，礦物、分解、風化皆非主要來源。(課本對應)
29.關於能量塔（能量金字塔），下列敘述何者正確？能量流動
A最高階消費者擁有的可用能量最多
B每升高一階能量會增加為原來的 10 倍
C每升高一個營養階層，可用能量約只剩前一階的 10%
D能量可在各營養階層間循環使用
【答案】C　【解析】energy：能量沿食物鏈單向流動，每升一階約只剩 10%，其餘約 90% 以呼吸與熱散失。能量不循環，底寬頂窄，最高階能量最少。(課本對應)

A6000

D60

31.下列關於生態系中各角色的敘述，何者正確？能量流動
A消費者能行光合作用自製養分
B分解者不參與物質循環
C分解者（細菌、真菌）能分解遺體與排泄物，使物質回歸環境
D生產者必須攝食其他生物才能獲得能量
【答案】C　【解析】energy：分解者分解遺體與排泄物、使物質回歸環境，是物質循環關鍵。生產者自製養分、消費者攝食他物、分解者參與循環，其餘選項錯誤。(課本對應)
32.在碳循環中，下列哪一作用能「減少」大氣中的二氧化碳？物質循環
A燃燒化石燃料
B分解作用
C光合作用
D呼吸作用
【答案】C　【解析】cycle：光合作用固定 CO₂ 是減少大氣 CO₂ 的主要途徑（另有溶於海水）。呼吸、燃燒、分解都會釋放 CO₂ 而使其增加。(課本對應)
33.關於水域的「優養化」現象，下列敘述何者正確？物質循環
A氮、磷等營養鹽過多使藻類大量繁殖，最終造成水中溶氧大減、魚蝦死亡
B營養鹽過多會使水中溶氧持續增加
C優養化能提升水質、有利養殖
D優養化是因水溫過低結冰造成
【答案】A　【解析】cycle：優養化是氮磷過多→藻類爆增（藻華）→遮光與分解耗氧→溶氧大減→魚蝦死亡、水質惡化。它使溶氧減少、水質變差，與結冰無關。(課本對應)

35.關於「生物圈」的敘述，下列何者正確？生態系與生物圈
A生物圈範圍永遠固定不變，不會隨生物滅絕而改變
B生物圈是地球上所有生物及其生存環境的總和，範圍約海平面上下各約 1 萬公尺

D月球表面也屬於地球生物圈的一部分
【答案】B　【解析】eco：生物圈是所有生物與其環境的總和，範圍約海平面上下各約 1 萬公尺。它會隨生物發現或滅絕而改變、分布不均、不含月球，且佔地球比例極小。(課本對應)

41.生物多樣性包含三個層次，下列關於三層次的敘述何者正確？多樣性與保育
A遺傳多樣性指同一物種內個體間的基因或性狀差異
B遺傳多樣性指一地區內所有物種的種類豐富度
C生態系多樣性指某一物種能適應的棲地數量
D物種多樣性指同一物種內不同族群的數量
【答案】A　【解析】conserv：遺傳（基因）多樣性是同種內的基因/性狀變異，如同種綠豆有不同品系、人類膚色不同。物種多樣性是種類豐富度，生態系多樣性是棲地/生態系類型多寡，選項把三者定義混淆。(課本對應)
42.同一品種的玫瑰，因雜交與突變而開出紅、粉、黃、白等不同花色，這主要展現了生物多樣性的哪一個層次？多樣性與保育
A物種多樣性
B生態系多樣性
C遺傳（基因）多樣性
D群集多樣性
【答案】C　【解析】conserv：同一物種內出現不同性狀（花色）是同種基因變異，屬遺傳多樣性。不同種類才是物種多樣性，不同棲地才是生態系多樣性。(課本對應)
43.關於 IUCN（國際自然保育聯盟）所編製的紅皮書（紅色名錄），下列敘述何者正確？多樣性與保育

B它即為華盛頓公約的別名

D它依滅絕風險將物種分級，用以評估物種的存續狀況
【答案】D　【解析】conserv：IUCN 紅色名錄評估物種滅絕風險並分級（EX、EW、CR、EN、VU…）。管制國際貿易的是 CITES（華盛頓公約），保護臭氧層的是蒙特婁議定書，紅色名錄並非華盛頓公約。(課本對應)
